Overview

The Email MIME Parser is a robust and efficient **Java library** designed for parsing complex email messages encoded in MIME (Multipurpose Internet Mail Extensions) format. It provides a straightforward API to extract various parts of an email, including text content, HTML content, attachments, and intricate header information.

Whether you're building an email client, an archival system, or an analytics tool, this library simplifies the often challenging task of decoding and understanding email structures.

Features

  • Comprehensive MIME Parsing: Handles multi-part messages, including mixed, alternative, related, and digest types.
  • Content Extraction: Easily retrieve plain text and HTML body parts.
  • Attachment Handling: Identify and extract file attachments with their original filenames and content types.
  • Header Parsing: Access all email headers (e.g., From, To, Subject, Date, Content-Type, Content-Transfer-Encoding) in a structured manner.
  • Character Encoding Support: Decodes various character sets commonly found in email.
  • Robust Error Handling: Designed to gracefully handle malformed or non-standard MIME messages.
  • Pure Java: Written entirely in Java, ensuring cross-platform compatibility and ease of integration into Java projects.

Installation

You can add the Email MIME Parser to your Java project using Maven or Gradle.

Maven

Add the following dependency to your pom.xml:

<dependency>
    <groupId>com.yourorganization.emailparser</groupId> <!-- Replace with actual GroupId -->
    <artifactId>email-mime-parser</artifactId>
    <version>1.0.0</version> <!-- Replace with the latest version -->
</dependency>

Gradle

Add the following to your build.gradle file:

implementation 'com.yourorganization.emailparser:email-mime-parser:1.0.0' // Replace with actual GroupId and latest version

Alternatively, you can download the JAR directly from the GitHub releases page (if available) and add it to your project's classpath.

Usage

Here's a quick example to get started with parsing an email message:

import com.yourorganization.emailparser.EmailParser; // Adjust package name
import com.yourorganization.emailparser.ParsedEmail; // Adjust package name
import com.yourorganization.emailparser.Attachment;  // Adjust package name

import java.io.ByteArrayInputStream;
import java.io.InputStream;
import java.nio.charset.StandardCharsets;
import java.util.Map;

public class EmailParsingExample {

    public static void main(String[] args) {
        String emailContent = "From: Sender <sender@example.com>\n" +
                              "To: Recipient <recipient@example.com>\n" +
                              "Subject: Test Email with Attachment\n" +
                              "MIME-Version: 1.0\n" +
                              "Content-Type: multipart/mixed; boundary=\"BOUNDARY_STRING\"\n" +
                              "\n" +
                              "--BOUNDARY_STRING\n" +
                              "Content-Type: text/plain; charset=\"utf-8\"\n" +
                              "Content-Transfer-Encoding: quoted-printable\n" +
                              "\n" +
                              "Hello, this is the plain text part of the email.\n" +
                              "=0ADon't forget the meeting.\n" +
                              "\n" +
                              "--BOUNDARY_STRING\n" +
                              "Content-Type: text/html; charset=\"utf-8\"\n" +
                              "Content-Transfer-Encoding: quoted-printable\n" +
                              "\n" +
                              "<html><body><p>Hello, this is the <b>HTML part</b> of the email.</p></body></html>\n" +
                              "\n" +
                              "--BOUNDARY_STRING\n" +
                              "Content-Type: text/plain; name=\"attachment.txt\"\n" +
                              "Content-Disposition: attachment; filename=\"attachment.txt\"\n" +
                              "Content-Transfer-Encoding: base64\n" +
                              "\n" +
                              "SGVsbG8gZnJvbSB0aGUgYXR0YWNobWVudCE=\n" +
                              "\n" +
                              "--BOUNDARY_STRING--";

        InputStream emailStream = new ByteArrayInputStream(emailContent.getBytes(StandardCharsets.UTF_8));

        try {
            // Parse the email
            ParsedEmail email = EmailParser.parse(emailStream);

            System.out.println("--- Email Headers ---");
            for (Map.Entry<String, String> header : email.getHeaders().entrySet()) {
                System.out.println(header.getKey() + ": " + header.getValue());
            }

            System.out.println("\n--- Email Body (Plain Text) ---");
            if (email.getTextBody() != null && !email.getTextBody().isEmpty()) {
                System.out.println(email.getTextBody());
            } else {
                System.out.println("No plain text body found.");
            }

            System.out.println("\n--- Email Body (HTML) ---");
            if (email.getHtmlBody() != null && !email.getHtmlBody().isEmpty()) {
                System.out.println(email.getHtmlBody());
            } else {
                System.out.println("No HTML body found.");
            }

            System.out.println("\n--- Attachments ---");
            if (email.getAttachments() != null && !email.getAttachments().isEmpty()) {
                for (Attachment attachment : email.getAttachments()) {
                    System.out.printf("Filename: %s, Content-Type: %s, Size: %d bytes\n",
                        attachment.getFilename(), attachment.getContentType(), attachment.getContent().length);
                    // To save attachment: Files.write(Paths.get(attachment.getFilename()), attachment.getContent());
                }
            } else {
                System.out.println("No attachments found.");
            }

        } catch (Exception e) {
            System.err.println("Error parsing email: " + e.getMessage());
            e.printStackTrace();
        }
    }
}
